# Backend - Food Delivery Project

The **Backend** is built with **Node.js and Express.js** and handles all API requests for the Food Delivery application.  

## Features

- User authentication (register/login)
- Manage food items (CRUD operations)
- Handle cart and orders
- Connects with MongoDB database

## Setup

#### 1. Install dependencies:
```bash
npm install
```

#### 2. Create a .env file with your environment variables :
```bash
JWT_SECRET = "your_jwt_secret"
MONGODB_URI = "your mongodb_connection_string"
```

#### 3. Run the backend server:
```bash
npm run server
```
