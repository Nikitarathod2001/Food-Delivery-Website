# Frontend - Food Delivery Project

The **Frontend** is built with **React.js** and provides a user-friendly interface for the Food Delivery project.  

## Features

- User registration and login
- Add food items to cart
- Remove items from cart
- Place orders

## Setup

#### 1. Install dependencies:
```bash
npm install
```

#### 2. Start the frontend server:
```bash
npm run dev
```
