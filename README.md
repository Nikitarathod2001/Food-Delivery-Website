# Food Delivery Project

A full-stack **Food Delivery Web Application** where users can create an account, log in, add food items to their cart, and place orders.  

This project is divided into three main modules: `admin`, `backend`, and `frontend`.  

## Features

- User registration and login
- Add and remove food items
- Cart functionality with order placement
- Full-stack architecture (Node.js, Express, MongoDB, React)


## Instructions

## 1. Clone the repository:
```bash
git clone https://github.com/Nikitarathod2001/Food-Delivery-Website.git
```

## 2. Navigate to each folder and install dependencies:
```bash
cd admin
npm install

cd backend
npm install

cd frontend
npm install
```

## 3. Run the applications:
- backend
```bash
cd backend
npm start
```

- frontend
```bash
cd frontend
npm start
```
